Setup YouTube API

Note: Your API key remains on your device and is only used to fetch additional metadata about your watched videos.

How to Get Your YouTube API Key

Step 1: Create a Google Cloud Project

Go to the Google Cloud Console and create a new project or select an existing one.

Step 2: Enable the YouTube Data API

Search for the "YouTube Data API v3", and click it.

Step 3: Enable API

Click the "Enable" button to activate the YouTube Data API for your project.

Step 4: Navigate to Credentials

In the left-hand menu, click on "Credentials" to proceed.

Step 5: Create Credentials

Click on "Create credentials" and select "API key". This will generate a new API key for you to use.

Step 6: Copy API key

Copy the API key and put it at the top of this page and click Save.
